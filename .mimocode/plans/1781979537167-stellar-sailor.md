# Plan: Add Charts to Admin Reports Page

## Goal
Add 4 interactive charts to `admin/reports.php` using **Chart.js** (loaded via CDN). No new npm/composer deps needed.

## Charts to Add
1. **Player Gender Pie Chart** — male vs female vs other distribution
2. **Tournament Format Pie Chart** — single_elimination vs round_robin vs double_elimination
3. **Tournament Status Doughnut** — upcoming vs ongoing vs completed vs cancelled
4. **Win/Loss Ratio Doughnut** — total wins vs total losses across all players

## Approach

### 1. Add Chart.js CDN to footer.php (or inline in reports.php)
- Load Chart.js 4.x via `<script src="https://cdn.jsdelivr.net/npm/chart.js"></script>`
- Since footer.php has `$extraJs` support, we'll set `$extraJs` in reports.php to load Chart.js only on this page

### 2. Modify `admin/reports.php`
**PHP changes** — add new data queries at top:
- Gender distribution: `SELECT gender, COUNT(*) FROM players GROUP BY gender`
- Format distribution: `SELECT format, COUNT(*) FROM tournaments GROUP BY format`
- Status distribution: `SELECT status, COUNT(*) FROM tournaments GROUP BY status`
- Win/Loss: already computed as `$totalWins` / `$totalLosses`

**HTML changes** — add a new `<div class="content-grid">` section with 4 canvas elements in a 2x2 grid layout, placed between the stats cards and the existing tables.

**JS changes** — add `<script>` block at bottom rendering all 4 charts with the PHP data passed as JSON via `json_encode()`.

### 3. Styling
- Use existing `.content-grid` and `.card` CSS classes for consistency
- Charts will be responsive via Chart.js `responsive: true`
- 2-column grid: 2 charts per row

## Files to Modify
- `C:\xampp\htdocs\TournamentHQ\admin\reports.php` — add queries, chart HTML, and Chart.js script

## Verification
1. Log in as admin → navigate to Reports page
2. Verify all 4 charts render correctly with data
3. Verify charts are responsive on smaller screens
4. Verify existing tables/stats still display correctly
