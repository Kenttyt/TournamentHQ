# Live WebSocket Scoring — Implementation Plan

## Architecture

```
Score POST → recordBracketMatchResult() → writes signal file → WS server detects → broadcasts → clients AJAX-fetch fresh bracket HTML → DOM replace
```

- **WS Server**: Standalone PHP CLI script (`scripts/websocket_server.php`), port 9000
- **IPC**: Signal files in `var/ws_signals/` (zero dependencies, no Redis/Composer)
- **Client refresh**: AJAX fetch from new endpoint → replace bracket DOM → re-execute connector lines

---

## New Files (3)

### 1. `scripts/websocket_server.php` (~250 lines)
Hand-rolled RFC 6455 WebSocket server:
- `handshake()` — read HTTP Upgrade header, compute `Sec-WebSocket-Accept`, send 101
- `decodeFrame()` / `encodeFrame()` — WebSocket framing (opcode 0x1 text, mask handling)
- Main loop: `socket_select()` with 500ms timeout
  - Accept new connections
  - Read client messages → parse `{"type":"join","tournament_id":N}` → add to `$rooms[$tid][]`
  - Detect disconnects → remove from room
  - Poll `var/ws_signals/*.json` → broadcast `{"type":"score_update","tournament_id":N}` to room → delete file
- Heartbeat ping every 30s
- Configurable via `--port=9000` CLI arg

### 2. `scripts/run_websocket.bat`

### 3. `includes/bracket_view_ajax.php` (~25 lines)
Lightweight AJAX endpoint returning bracket HTML fragment (no header/footer/sidebar):
- Validates `tournament_id` GET param
- Requires auth (logged-in users only)
- Calls `buildBracketGroups($tid)`, sets `$recordResultUrl = ''` (read-only display)
- Includes `bracket_view.php` → returns pure bracket HTML

## Modified Files (3)

### 4. `modules/matches/match_functions.php`
**Add `broadcastScoreUpdate()` function** (after line 436):
```php
function broadcastScoreUpdate(int $tournamentId): void {
    $dir = __DIR__ . '/../../var/ws_signals';
    if (!is_dir($dir)) @mkdir($dir, 0777, true);
    $file = $dir . '/score_' . $tournamentId . '_' . time() . '_' . substr(md5(uniqid('', true)), 0, 8) . '.json';
    @file_put_contents($file, json_encode(['tournament_id' => $tournamentId]));
}
```
