# Team Event Support

## Summary
Add a global "Team Event" toggle to tournaments. When enabled, registration forms show a single "Team Name" field instead of First/Last Name fields.

---

## 1. Database

### New column on `tournaments`: `is_team_event`
```sql
is_team_event TINYINT(1) NOT NULL DEFAULT 0
```

Migration in `config/database.php` — same try/catch pattern as other columns.

---

## 2. Files to Modify

### `config/database.php`
- Add migration for `tournaments.is_team_event` column.

### `modules/tournaments/tournament_functions.php`
- `createTournament()` — include `is_team_event` in INSERT.
- `updateTournament()` — include `is_team_event` in UPDATE.

### `organizer/tournaments.php`
- **Create form**: Add checkbox "Team Event" next to sport/category row.
- **Edit form**: Add checkbox, pre-select from `t.is_team_event`.
- **POST handlers**: Capture `is_team_event` from `$_POST`.
- **Open edit modal JS**: Set checkbox state.

### `admin/manage_tournaments.php`
- Same changes as organizer: create form, edit form, POST handler, JS.

### `player/index.php`
- **Registration modal**: When tournament `is_team_event = 1`:
  - Replace First Name / Last Name fields with a single "Team Name" field.
  - Change placeholder text to "Enter each team."
  - JS `addGuestRow()` checks tournament flag to render correct fields.
- **POST handler**: When team event, store team name in `first_name`, leave `last_name` empty.
- **Display registered players**: Show team name (`first_name`) instead of full name.

### `organizer/tournaments.php` (player list display)
- When `is_team_event`, display `first_name` (team name) instead of `first_name + last_name`.

### `organizer/players.php`
- Same display logic for team names.

### `play.php` (public listing)
- No changes needed (doesn't show individual players).

---

## 3. UI Design

### Tournament Create/Edit Form
```
[ Tournament Name ] [ Sport ▾ ] [ Category Name ]
[ ] Team Event — check this if participants register as teams (e.g. doubles, team relay)
```

### Player Registration Modal (Team Event ON)
```
Teams to register *
[ Team Name ]
[ + Add Team ]

Enter each team that will play.
```

### Player Registration Modal (Team Event OFF — unchanged)
```
Players to register *
[ First Name ] [ Last Name ]
[ + Add Player ]
```

---

## 4. Data Storage Strategy

For team events, store the team name in `tournament_guests.first_name` and leave `last_name` empty. This avoids schema changes to the guests table. Display logic checks `is_team_event` to decide how to render names.

---
